## 中后台系统业务的特点

- 工作量大、页面多
- 交付复杂
- 页面信息量大
- 涉及数据模型复杂
- 工作效率要求大于审美要求
- 多数情况下属于刚需使用

质量的衡量体系和PV/UV不太一样

![](https://moonstarimg.oss-cn-hangzhou.aliyuncs.com/picgo_img/20210922091440.png)

## 中后台体系的目标

- 开发效率（复用性）
- 易用性效率>美观、体验
- 便捷的交互
- 复杂数据处理能力
- 可访问性要求提升
- 国际化要求提升
